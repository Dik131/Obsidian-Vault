# –ü–∞–º—è—Ç—å –∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ –∫–æ–¥–∞ –≤ JavaScript

**Browser or Web APIs** ‚Äî —ç—Ç–æ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å—ã, –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ –≤ –≤–µ–±-–±—Ä–∞—É–∑–µ—Ä, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—é—Ç –¥–æ—Å—Ç—É–ø –∫ –¥–∞–Ω–Ω—ã–º –±—Ä–∞—É–∑–µ—Ä–∞ –∏ –æ–∫—Ä—É–∂–µ–Ω–∏—é –∫–æ–º–ø—å—é—Ç–µ—Ä–∞, –ø–æ–∑–≤–æ–ª—è—è –≤—ã–ø–æ–ª–Ω—è—Ç—å —Å–ª–æ–∂–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏. –û–Ω–∏ –Ω–µ —è–≤–ª—è—é—Ç—Å—è —á–∞—Å—Ç—å—é —Å–∞–º–æ–≥–æ —è–∑—ã–∫–∞ JavaScript, –∞ –ø–æ—Å—Ç—Ä–æ–µ–Ω—ã –ø–æ–≤–µ—Ä—Ö –Ω–µ–≥–æ, –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—è –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏ –¥–ª—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –≤ –∫–æ–¥–µ JavaScript.

## –û—á–µ—Ä–µ–¥—å —Å–æ–±—ã—Ç–∏–π –∏–ª–∏ –∫–æ–ª–±—ç–∫–æ–≤ (Event or Callback Queue)
